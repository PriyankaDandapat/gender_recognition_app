Welcome to Face Recognition Web App project !

This webapp takes .jpg, .jpeg and .png format files and predicts the gender the person in the photo.

Technologies Used :-

Sr. No. Name Used for
1 Haar Cascade Classifier (XML) Used for face detection in images or video frames. The Haar cascade classifier is a popular algorithm for facial detection due to its computational efficiency and high accuracy.
2 Principal Component Analysis (PCA) Used for dimensionality reduction of face images. PCA helps reduce the number of features while preserving most of the information in the dataset
3 Support Vector Machine (SVM) Used for gender prediction based on the extracted features. SVM is a machine learning algorithm that can classify data into different categories, making it suitable for gender classification.
4 Flask Used as a web framework to build the face recognition web app. Flask is a lightweight and flexible framework for creating web applications in Python.
5 OpenCV Used for image processing tasks, such as loading images, applying filters, and performing face detection. OpenCV is a popular computer vision library that provides a wide range of functions and algorithms for image and video analysis.
6 Pandas Used for data manipulation and analysis. Pandas is a powerful library for handling structured data, making it useful for tasks such as loading and preprocessing datasets.
7 Matplotlib Used for data visualization. Matplotlib is a plotting library that allows you to create various types of charts and graphs to visualize data.
8 NumPy Used for numerical computations. NumPy is a fundamental library for scientific computing in Python, providing support for large, multi-dimensional arrays and a collection of mathematical functions.
